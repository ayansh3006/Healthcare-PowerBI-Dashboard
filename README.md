# 🏥 Health Data Analytics Dashboard | Power BI

An interactive **Power BI dashboard** designed to analyze healthcare data, patient demographics, hospital operations, treatment trends, and billing information. This project transforms raw healthcare data into actionable insights through interactive dashboards, enabling data-driven decision-making.

---

## 📌 Project Overview

The **Health Data Analytics Dashboard** provides a comprehensive view of hospital and patient data using interactive visualizations. It enables users to monitor patient demographics, admission trends, treatment costs, and hospital performance through intuitive reports.

The dashboard is organized into three key report pages:

- 👥 Patient Demographics
- 📈 Healthcare Trends
- 💰 Treatment & Cost Analysis

---

- Analyze patient demographic information.
- Monitor hospital admissions and treatment trends.

## 📊 Dashboard Features

### 👥 Patient Demographics

Provides an overview of patient information and hospital resources.

**KPIs**
- Total Patients
- Average Patient Age
- Average Length of Stay (LOS)
- Total Doctors
- Total Rooms
- Average Billing Amount

**Visualizations**
- Patient Distribution by Age Group
- Blood Group Distribution
- Medical Condition Analysis
- Gender Distribution
- Interactive Slicers

---

### 📈 Healthcare Trends

Analyzes patient admissions and healthcare trends over time.

**Visualizations**
- Monthly Admission Trends
- Trend Analysis
- Donut Charts
- Comparative Charts
- Interactive Filters

---

### 💰 Treatment & Cost Analysis

Focuses on treatment expenses and hospital billing.

**Includes**
- Treatment-wise Billing Analysis
- Average Treatment Cost
- Expense Comparison
- Cost Distribution
- Interactive Tables

---

## 📈 Key Insights

- Understand patient demographics quickly.
- Monitor hospital resource utilization.
- Analyze admission trends over time.
- Compare treatment and billing costs.
- Identify common medical conditions.
- Support operational and strategic decision-making.
